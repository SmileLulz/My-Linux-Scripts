# Dependency

- No critical dependencies; only uses `curl`

# Usage and Arguments

- Usage:
  - `mcfetch rp-formats`   (fetch resource pack formats)
  - `mcfetch dp-formats`   (fetch datapack formats)

- Arguments:

  - `-s` or `--snapshots`  (include snapshots)

  - `-v` or `--version`    (only fetch for specific version; e.g. 1.20.x or 1.21.4)
