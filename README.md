# Description

I wrote some scripts for my personal use. I'm publishing it publicly so that newbies can use too :)

And don't ask for updates pls :)

---

# Insteallation

Put any script you like in your `~/.local/bin` folder.

# Scripts Info

- [playyt](https://github.com/SmileLulz/My-Linux-Scripts/tree/main/playyt) : Play YouTube audios from your terminal without locking the session
- [mcfetch](https://github.com/SmileLulz/My-Linux-Scripts/tree/main/mcfetch) : Fetch Minecraft (currently java only) resource pack and datapack formats (versions)
- [konsole](https://github.com/SmileLulz/My-Linux-Scripts/tree/main/konsole) : Open KDE Konsole terminal in new tab instead of new window
